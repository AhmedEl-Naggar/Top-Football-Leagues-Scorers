This project is an interactive Football Analytics Dashboard built using Power BI, designed to analyze player, club, league, and country performance across multiple football metrics.
The dashboard provides deep insights into shooting, attacking, finishing, and playing time using advanced performance indicators such as xG, shot accuracy, goal conversion, and availability rate.

It is aimed at data analysts, football analysts, scouts, and decision-makers who want to evaluate performance based on data-driven metrics.
